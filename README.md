Lenovo K3 Note 
==============

This branch is for building of the LineageOS 14.1 and all Lineage Based Roms

# Build Information

### Working:
 * Wi-Fi
 * Wi-Fi tethering
 * GPS
 * Bluetooth
 * AGPS
 * Vibration
 * Audio
 * Auto brightness
 * RAM and ROM
 * Rotation
 * All sensors
 * Flashlight (statusbar and camera)
 * Camera rear/front (video and photo)
 * Micro SD support
 * RIL for SMS/calls/internet/LTE 
 * MTP 
 * LED Notifications
 * FM Radio

### Not Working:

 * Offline charging
 * Camera HDR
